## 상태 짧게 확인하기
- git status -s
  XY
  - x: staging area의 상태
  - y: 현재 디렉토리의 상태
  - M

- ?? : untracked (새로 생성된 파일)
- A : added
- M : modified
- R : renamed
- MM
- AM :
- !! : ignored (깃에 보이지 않는 파일)
  -- ignored : 깃에 보이지 않는 파일의 상태까지 나타냄
- "" : clean (아무것도 안나타나면 클린한 상태)
